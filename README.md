# k8s-health-toolkit
Comprehensive Kubernetes and OpenShift health check, diagnostics, and reporting.

## Scripts
| Script | Purpose |
|--------|---------|
| `scripts/cluster_health.sh` | Full cluster health check |
| `scripts/pod_troubleshoot.sh` | Deep-dive pod diagnostics |
| `scripts/resource_usage.sh` | Node and namespace resource consumption |
| `scripts/certificate_check.sh` | Check K8s certificate expiry |
| `scripts/etcd_health.sh` | etcd cluster health and performance |
